Android tutorials
=================

Use these tutorials to quickly voice enable Android apps written in Java and Kotlin:

- `Building a voice assistant for an Android Java or Kotlin app <integrating-java-kotlin.html>`__
   
  Learn how create a voice interface for an Android Java or Kotlin app from scratch and write simple voice commands.
	 
- `Navigating between screens in an Android app (Kotlin) <navigating-kotlin.html>`__
   
  Learn how to send commands to an iOS Swift app, handle commands on the app side and navigate between screens with voice.

- `Passing the app state to the voice script <sending-data.html>`__
   
  Learn how to pass the app state from an Android app to the voice script with visual state, access the data passed with visual state in the voice script and filter intents by the app state.
	 
- `Highlighting items with voice (Kotlin) <highlighting-kotlin.html>`__
   
  Learn how to go through the list of items with voice and highlight the current item in the app UI.
