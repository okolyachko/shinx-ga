

Getting started
===============

   <div id="main-page"></div>

   <p style="font-size: 20px">Want to create an in-app voice assistant? <br/>Follow the steps below to get started with the Alan Platform.</p>

   .. raw:: html
	
	  <p style="font-size: 28px; font-weight: 700">1. Create an Alan Studio account</p>
	  
   Sign up for Alan Studio, Alan's web-based IDE, to create a voice assistant project, test the dialog and customize your voice assistant.

      <p style="font-size: 28px; font-weight: 700">2. Write a voice script</p>
	  
   In Alan Studio, write a voice script. Voice scripts describe the structure of the dialog with the user, or the expected conversation scenario.

   Get started swiftly:
   
	
   - :doc:`Go to Script concepts <server-api/script-concepts>` to learn what Alan's functionality you can use to create a voice script.

   - `Check out Example scripts <https://github.com/alan-ai/voice-assistant-scripts>`__ in Alan Studio to see how voice scripts can be organized.
   
   
   
.. raw:: html
   
   <p style="font-size: 28px; font-weight: 700; text-align:center">3. Integrate a voice assistant with your app</p>
   
Alan offers SDKs for different platforms to embed a voice assistant to your app. Select your platform to learn about :doc:`integration details <client-api/client-sdks>`. Upon integration, you'll see a small draggable voice button on top of the app's UI.

	  <p style="font-size: 28px; font-weight: 700">4. Analyze and iterate</p>
	  
   Alan provides you with advanced conversational analytics and success metrics through its :doc:`Logs <usage/guides/logs>` and :doc:`Analytics View<usage/guides/analytics-view>`. Use these tools to:

   - Gain insight into the users' experience
   - Analyze their behavior and flows
   - Identify unhandled requests 
   - Optimize the voice assistant effectiveness
