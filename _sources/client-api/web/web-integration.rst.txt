Integration with Web projects
=============================

To integrate Alan with a web project, you can use the following SDKs:

- `Alan Web SDK <web-api>`__ to integrate with:

  - `React <react>`__
  - `Angular <angular>`__
  - `Vue <vue>`__
  - `Ember <ember>`__
  - `JavaScript <vanilla>`__
  - `Electron <electron>`__

- `Alan Web component <web-component>`__ to integate with cross-platform frameworks like `Ionic <../cross-platform/ionic>`__.
