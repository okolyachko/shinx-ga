Integration with Web projects
=============================

To integrate Alan with a web project, you can use the following SDKs:

- :doc:`Alan Web SDK <web-api>` to integrate with:

  - :doc:`React <react>`
  - :doc:`Angular <angular>`
  - :doc:`Vue <vue>`
  - :doc:`Ember <ember>`
  - :doc:`JavaScript <vanilla>`
  - :doc:`Electron <electron>`

- :doc:`Alan Web component <web-component>` to integate with cross-platform frameworks like :doc:`Ionic <../cross-platform/ionic>`.
